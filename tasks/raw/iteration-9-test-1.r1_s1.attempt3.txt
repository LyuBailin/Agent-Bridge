# 个人笔记管理系统 (Personal Notes Management System)

## 项目描述

这是一个简单的个人笔记管理系统，允许用户创建、编辑和管理笔记。系统使用Node.js和Express作为后端框架，并使用SQLite数据库来存储数据。

## 功能列表

- 用户注册和登录
- 创建、编辑和删除笔记
- 搜索和分类笔记
- 用户个人信息管理

## 安装说明

1. 确保已安装Node.js和npm。
2. 克隆项目仓库：
   ```bash
   git clone <repository-url>
   ```
3. 进入项目目录：
   ```bash
   cd personal-notes-management-system
   ```
4. 安装依赖：
   ```bash
   npm install
   ```
5. 配置SQLite数据库：
   - 确保SQLite已安装。
   - 创建数据库文件 `db.sqlite`。

## 使用指南

1. 启动服务器：
   ```bash
   npm start
   ```
2. 打开浏览器，访问 `http://localhost:3000`。
3. 注册并登录账户。
4. 开始创建和管理笔记。

## 技术栈

- 后端：Node.js, Express
- 数据库：SQLite

## 部署信息

1. 将项目代码部署到服务器上。
2. 确保Node.js和npm已安装。
3. 安装依赖：
   ```bash
   npm install --production
   ```
4. 启动服务器：
   ```bash
   node app.js
   ```
