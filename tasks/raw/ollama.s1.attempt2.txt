And for file operations:

To move the calculator.js and greeter.js files to a new lib/ directory:

```op
MV: lib/calculator.js -> ../lib/calculator.js
MV: lib/greeter.js -> ../lib/greeter.js
RM: app.js
```

Please note that the MV operation will automatically create parent directories if they do not exist, and RM operation is used to delete a file. The path in MV operation should be relative to the workspace root. In this case, we are moving calculator.js and greeter.js two levels up from the current directory (the one containing lib/calculator.js and lib/greeter.js) to the lib directory at the same level as app.js which will become a new root file app.js in the updated workspace context.
