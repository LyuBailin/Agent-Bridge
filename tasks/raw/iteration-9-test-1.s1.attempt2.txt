# 个人笔记管理系统

这是一个用于管理个人笔记的系统。

## 功能特点
- 用户注册和登录
- 笔记创建、编辑和删除
- 笔记分类和标签
- 笔记搜索功能

## 技术栈
- 前端: React
- 后端: Node.js, Express
- 数据库: MongoDB

## 安装步骤
1. 克隆仓库
2. 安装依赖
   ```bash
   npm install
   ```
3. 配置环境变量
4. 启动服务
   ```bash
   npm start
   ```
