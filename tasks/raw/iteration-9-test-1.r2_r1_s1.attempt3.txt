# 个人笔记管理系统

## 项目标题
个人笔记管理系统

## 简介
这是一个基于Node.js, Express和SQLite构建的简单个人笔记管理系统。用户可以创建、编辑、查看和删除笔记。

## 功能特性列表
- 用户注册与登录
- 创建、编辑和删除笔记
- 笔记分类和搜索功能

## 技术栈
- Node.js
- Express
- SQLite

## 安装步骤
1. 确保已安装Node.js和npm。
2. 克隆项目仓库到本地：
   ```bash
   git clone <repository-url>
   ```
3. 进入项目目录：
   ```bash
   cd 个人笔记管理系统
   ```
4. 安装依赖包：
   ```bash
   npm install
   ```
5. 启动数据库：
   ```bash
   npx sequelize-cli db:migrate
   ```
6. 启动服务器：
   ```bash
   npm start
   ```

## 使用指南
1. 打开浏览器，访问`http://localhost:3000`。
2. 注册或登录账户。
3. 创建新的笔记，并输入标题和内容。
4. 编辑或删除已有的笔记。

## 部署说明
1. 确保服务器上已安装Node.js和npm。
2. 将项目代码上传到服务器。
3. 安装依赖包：
   ```bash
   npm install --production
   ```
4. 启动数据库：
   ```bash
   npx sequelize-cli db:migrate
   ```
5. 配置环境变量，如数据库连接字符串。
6. 启动服务器：
   ```bash
   npm start
   ```
